# GLfinal

可更動code <br/>
main.cpp, shader code <br/>

控制 <br/>
WASD: 移動 <br/>
滾輪: 控制移動速度 <br/>
F: 切換Post Effect效果 <br/> 
T: Normal mapping On/Off (只有在橡樹有效) <br/> 

效果 <br/>
SSAO與Normal mapping只在原點橡樹有效 <br/>
deferred rendering的multi light只在非原點橡樹有效 <br/>
